# Screenshots Directory

This directory contains screenshots for the RestToRun documentation.

## File Names

Please name your screenshots according to the following convention:

- `date-range-selector.png` - Date range selection UI
- `sleep-breakdown-chart.png` - Sleep breakdown visualization
- `training-metrics-chart.png` - Running metrics chart
- `weekly-mileage-chart.png` - Weekly mileage comparison chart
- `goals-and-progress.png` - Goals & Progress section
- `recovery-metrics.png` - Recovery metrics statistics
- `performance-metrics.png` - Performance metrics statistics
- `training-load-analysis.png` - Training Load & Recovery Analysis
- `yearly-mileage-goals.png` - Yearly mileage planning interface

## Adding Screenshots

1. Take high-quality screenshots of each feature
2. Name them according to the convention above
3. Save them in this directory
4. Update FEATURES.md if the image paths need adjustment
